# murasanca.com

This repository contains the official website of Murat Sancak, known online as murasanca. It presents his professional identity as a computer engineer and software developer through an accessible, dependency-free, and fast static experience.

## Architecture

The website uses CSS, HTML, and JavaScript without an application framework, build process, package installation, or server-side runtime. Its directly addressable pages remain compatible with GitHub Pages and static hosting.

## Deployment

GitHub Pages serves the repository as a static website. The production custom domain remains intentionally inactive in this repository while the existing website is live.

## Development

Serve the repository root with a static HTTP server:

```sh
python3 -m http.server 4173
```

Open `http://localhost:4173` in a web browser.
